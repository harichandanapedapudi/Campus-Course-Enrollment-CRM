 Campus Course Enrollment & Student Support CRM

 Project Description
This Salesforce CRM project is designed to manage student enrollment, course completion, certificate issuance, and reporting using Salesforce declarative and programmatic features.

 Key Features
- Custom Objects: Student, Course, Certificate Request
- Automation using Flows and Apex
- Duplicate Management
- Reports and Dashboards
- Role-based Security
- Data Import and Validation

 Technologies Used
- Salesforce Developer Edition
- Apex Classes & Triggers
- Lightning Record Pages
- Reports & Dashboards
